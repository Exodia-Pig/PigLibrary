### 시작 동기
함수형 프로그래밍을 학습하고싶은 계기가 자꾸 생겨서 우선 코틀린으로 간단하게 알아보기위해 스터디 시작  
-> 마르친 모스카와 책이 얇고 구성 단위가 적어서 혼자서 데일리하게 책읽기 좋음  
- 컴포즈 등 최신 안드로이드 개발 주류 흐름에서 함수형의 개념이 자꾸 주류로 흘러들어옴
- 동시성 문제를 공부하다 함수형이 동시성을 풀수있는 좋은키임을 발견
