### 1줄 후기
정말 좋은책이고 객체지향의 스타트를 끊기 너무 좋은책이다.  
이책을 읽는다고 뭐 객체지향이 생활화 되고 그런것은 아니지만 객체지향 채찍을 맞기전 개념을 준비하기 아주 좋다.  
이걸 읽고 넥스트스텝이나 우테코처럼 객체지향을 강제로 주입하는 채찍한번 맞으면 객체지향의 맛은 볼 수 있다.
